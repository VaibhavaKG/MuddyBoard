# MuddyBoard 🌟

MuddyBoard is a small interactive playground built for my niece using HTML, CSS, and JavaScript.

It combines simple games, animations, sounds, and photo-based activities into a single offline-friendly web app designed for young children. Many of the activities use her own photos, making the experience feel familiar and engaging.

## Features

* 11 interactive play worlds
* Custom photo integration throughout the app
* Bubble popping, painting, music, puzzles, and memory games
* Gentle sounds generated in the browser
* Hidden parent controls with child lock options
* Installable as a Progressive Web App (PWA)
* Works offline after installation
* No accounts, no ads, and no backend required

## Running Locally

```bash
npm install
npm run dev
```

Then open:

```text
http://localhost:3000
```

## Tech Stack

* HTML
* CSS
* Vanilla JavaScript (ES Modules)
* Web Audio API
* Progressive Web App (PWA)

## Why I Built It

I originally built this for my niece, who had a habit of appearing next to my laptop whenever I was trying to get work done. What started as a simple distraction gradually grew into a collection of small games and activities centered around her photos and favorite interactions.

A fun side project, a playground for experimentation, and something she genuinely enjoys using.
